# О библиотеке

Добро пожаловать в **QtPySignals** — реактивное ядро и набор утилит, собранные **QtPy** так, чтобы состояние вело себя как живая система: ты меняешь одно значение — и всё, что от него зависит, обновляется само. Без ручных подписок, без “invalidate кэша”, без бесконечных `useMemo` и `memo` ради того, чтобы UI перестал дрожать.

Здесь нет UI-фреймворка и “магического” state-комбайна. Есть компактное ядро (**signals / computed / effects + планировщик**), над ним — инструменты для глубоких структур и списков (**SignalMap**) и маленькие помощники вроде **Active**, которые спасают модалки и условные куски DOM там, где React обычно заставляет воевать с перерендерами.

Документация читается как одна история: **Core → SignalMap → React → Patterns**. Берёшь только то, что нужно, и не платишь сложностью за весь мир сразу.
